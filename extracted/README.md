# Westphal+ One-Pager

Basiert auf HTML5-Boilerplate-Prinzipien (semantisches Markup, SEO-Meta-Tags, Accessibility) mit Vite als Build-Tool.

## Lokal starten

```bash
npm install
npm run dev
```

Öffnet auf http://localhost:5173

## Für Produktion bauen

```bash
npm run build
```

Erzeugt eine optimierte Version im `dist/`-Ordner, die du auf jeden Webserver hochladen kannst.

## Wichtiger Hinweis: Kundenstimmen

Die drei Testimonials im Abschnitt "Kundenstimmen" sind **Platzhalter** (M. K. / J. R. / S. T.),
keine echten Kundenaussagen. Bitte vor dem Live-Schalten durch echte, anonymisierte
Kundenrückmeldungen ersetzen – frei erfundene Erfahrungsberichte sind in Deutschland laut
§ 5 UWG als irreführende Werbung unzulässig.

## Struktur

- `index.html` – Inhalt & Struktur
- `src/style.css` – Design-Tokens (Farben, Typografie) & Styling
- `src/main.js` – kleine Interaktivität (aktuelles Jahr im Footer)
- `public/images/` – Logo & Fotos von Paul Westphal

## Texte/Inhalte anpassen

Alle Texte liegen direkt in `index.html` als gut lesbares, semantisches HTML – einfach per
Suchen & Ersetzen anpassen.
